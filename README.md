doctors-office-template
======
**Doctors office template** is a [Bootstrap 3.x](http://getbootstrap.com) template that can be used for doctors offices.

## Used projects
* Images by openclipart.org
* Bootstrap theme *cerulean* from Bootswatch
* Font Awesome Icons
* Ionicons Icons
